## Linux

1. First clone the repository into a directory of your choice

```bash
git clone https://github.com/eozden-wq/Punchline
cd punchline
```

2. Install dependencies

```bash
npm i
```

3. Set up .env file

```bash
touch .env
```

and add the following .env file:

```txt
PORT = 3000
NODE_PATH = .
```

4. Run!

```bash
npm run start
```
